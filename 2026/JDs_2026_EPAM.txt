Key Responsibilities


AWS Serverless Development

    Design and implement serverless applications using AWS Lambda and Python
    Build scalable integration workflows using AWS services such as Lambda, S3, SQS, SNS, and API Gateway
    Implement event-driven architectures for asynchronous processing

API Development & Integration

    Design and develop REST APIs using Python
    Integrate with internal and external APIs
    Build reusable API clients and integration modules
    Implement API authentication mechanisms (OAuth, API Keys, JWT)

File Processing & Data Integration

    Develop solutions for file ingestion, transformation, and delivery
    Process files from S3, SFTP, or external systems

    Handle large files and batch processing scenarios

AWS Integration Services

    AWS Lambda
    API Gateway

    SQS / SNS
    EventBridge
    Step Functions (optional)

Observability & Reliability

    Implement logging and monitoring using CloudWatch
    Build robust error handling and retry mechanisms
    Ensure high availability and scalability of integration services

Infrastructure & DevOps

    Work with Infrastructure as Code (Terraform / CloudFormation / CDK)
    Participate in CI/CD pipeline implementation
    Ensure secure and compliant deployment of AWS resources
