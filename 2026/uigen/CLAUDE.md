  # CLAUDE.md                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                               
  This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.                                                                                                                                                                       
                                                                                                                                                                                                                                                                               
  ## Setup and Installation                                                                                                                                                                                                                                                    
                  
  1. **Optional** Edit `.env` and add your Anthropic API key if you want to use the AI-powered component generation:                                                                                                                                                           
                  
      ```plaintext                                                                                                                                                                                                                                                             
      ANTHROPIC_API_KEY=your-api-key-here
      ```                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                               
      If you don't provide an API key, static code will be returned instead.                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                               
  2. Install dependencies and initialize the database:                                                                                                                                                                                                                         
                  
      ```bash                                                                                                                                                                                                                                                                  
      npm run setup
      ```                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                               
      This command will:                                                                                                                                                                                                                                                       
      - Install all dependencies                                                                                                                                                                                                                                               
      - Generate Prisma client                                                                                                                                                                                                                                                 
      - Run database migrations                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                               
  ## Running the Application                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                               
  ### Development                                                                                                                                                                                                                                                              
                  
  To start the development server, run:                                                                                                                                                                                                                                        
                  
  ```bash                                                                                                                                                                                                                                                                      
  npm run dev     
                                                                                                                                                                                                                                                                               
  Open http://localhost:3000 in your browser.                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                               
  Usage                                                                                                                                                                                                                                                                        
                  
  1. Sign up or continue as an anonymous user.                                                                                                                                                                                                                                 
  2. Describe the React component you want to create in the chat.
  3. View generated components in real-time preview.                                                                                                                                                                                                                           
  4. Switch to Code view to see and edit the generated files.                                                                                                                                                                                                                  
  5. Continue iterating with the AI to refine your components.                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                                               
  Features                                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                               
  - AI-powered Component Generation: Use Claude to generate React components based on descriptions.                                                                                                                                                                            
  - Live Preview: See changes in real-time as you describe the component.
  - Virtual File System: Generated code is not persisted to disk, ensuring a clean development environment.                                                                                                                                                                    
  - Syntax Highlighting and Code Editor: Enjoy a rich coding experience with syntax highlighting and integrated code editor.                                                                                                                                                   
  - Component Persistence: Registered users can save and export their generated components.                                                                                                                                                                                    
                                                                                                                                                                                                                                                                               
  Development Commands                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                               
  - npm run dev: Start the development server.                                                                                                                                                                                                                                 
  - npm run build: Build the application for production.
  - npm run start: Start the production server.                                                                                                                                                                                                                                
  - npm run test: Run unit tests (if any).                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                               
  Technology Stack                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                               
  - Framework: Next.js 15 with App Router                                                                                                                                                                                                                                      
  - JavaScript/TypeScript: React 19, TypeScript
  - Styling: Tailwind CSS v4                                                                                                                                                                                                                                                   
  - Database: Prisma with SQLite                                                                                                                                                                                                                                               
  - AI Services: Anthropic Claude AI and Vercel AI SDK                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                               
  Tips for Development                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                               
  - Ensure Node.js 18+ and npm are installed.                                                                                                                                                                                                                                  
  - Optionally provide an Anthropic API key in .env for AI-powered component generation.
  - Run npm run setup to install dependencies and initialize the database.
